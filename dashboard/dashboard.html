Plain-language takeaways generated from the numbers above — a starting point, not a substitute for reviewing actual resources.

How this works: EventBridge triggers a Lambda function daily → Lambda calls the Cost Explorer API → writes one CSV per day to S3 (raw/YYYY-MM-DD.csv) plus a manifest.json index → this page (served over HTTPS via CloudFront, with the S3 bucket kept private behind an Origin Access Control) fetches the manifest, pulls each CSV client-side, and renders everything below. No server, no database, no public bucket.
